Design

To help you with your design, think about using a GenAI tool like ChatGPT - with prompts like: "I need to create a information system design document covering key user types, key processes etc, can you please guide me through this design process."
You should also consider attaching any files that describe your community. The GenAI tool can then help with specifics based on your communities needs / use-cases etc.
