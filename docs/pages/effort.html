Reference Set

Following is a set of activities that we are using as part of our research into measuring effort and outcome, relating to skills.

1/ Mastering a Skateboard Trick
1.1/ Learner A spends 30min everyday for 2 weeks (7 hours) to master flipping the board.
1.2/ Learner B spends 1 hour to master flipping the board.
2/ Completing a Project
2.1/ Learner A completes the project in the allocated time with an average skill capacity of "Capable".
2.2/ Learner A completes the project in the allocated time with an average skill capacity of "Knowledgeable".
3/ Running 5kms
3.1/ Learner A completes the 5kms in 1 hour, the same as last time they did it.
3.2/ Learner A completes the 5kms in 2 hours, which 15min quicker than the last time.

Decentralised Identifiers (DIDs)

"URL-based identifiers (URIs) in use on the Web today (2019) require that the identifier be leased from an authority such as a Domain Name Registrar. A Decentralized Identifier (DID) is an identifier that does not need to be leased; its creation and use is possible without a central authority to manage it. The advent of Blockchains and Decentralized Ledger Technologies have led to other innovations that support this new type of decentralized URI. DIDs have various benefits over more traditional URIs." - W3C - More...
