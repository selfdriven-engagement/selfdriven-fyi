Apps

For Learners, Learning-Partners & Community Facilitators

The following apps are built, supported and hosted as cloud services by selfdriven.

All the Apps are compatible, serving as specific interfaces to the user's or community's data.
All the app user interface code and data schemas are open source, so you can also build and host your own apps.
The apps are currently being tested in pilot programs.
If you would like to be involved in this testing then please contact us.
