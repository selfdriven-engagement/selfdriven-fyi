A key component of the selfdriven Operating System is the Improvement Cycle.

The improvement cycle includes learner critical reflection.

The improvement cycle and associated project/task/action management components within the selfdriven app enable project based learning (PBL).

The improvement cycle can be used to conceptualise:

  • Darwinism "Survival of the fittest", as in how best fit into an ecosystem.
  • Playing the "Infinite" game.
  • Healthy/least-harmful ecosystems.
